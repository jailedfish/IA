# Task Manager (Консольное приложение)

Автор: **Меньшов Даниил**

## Краткое описание
Консольный менеджер задач с возможностью добавления, редактирования, удаления, фильтрации, сохранения в JSON и отмены действий. Реализован на Python с использованием ООП, шаблона MVC, очереди по приоритету и стека для undo.

## Как запустить
1. Убедитесь, что установлен Python 3.8+
2. Клонируйте репозиторий или скопируйте файлы проекта
3. Запустите `main.py`:
   ```bash
   python3 main.py
   ```
## Тестирование

Проект содержит набор автоматических тестов, написанных с использованием `pytest`.

1. Убедитесь, что установлен pytest:
   ```bash
   pip install pytest
   ```
2. Используя запустите тесты командой:
```shell
pytest -v tests
```
